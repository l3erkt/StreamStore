## 📖 Overview

StreamStore demonstrates an end-to-end event-driven architecture where customer orders are produced, streamed through Apache Kafka, processed by consumers, and visualized in real time using Streamlit.

---

## ✨ Features

- 📦 Real-time order event streaming
- ⚡ Apache Kafka messaging
- 🐳 Dockerized infrastructure
- 📊 Interactive Streamlit dashboard
- 🔄 Event-driven architecture
- 🧩 Modular Python components
